Инструкция по запуску:

Установить необходимые библиотеки:
$ pip install flask

Запустить демо:
$ python run.py

Открыть в браузере страницу:
http://127.0.0.1:5000/   (или же ту, которую предложит программа в выводе консоли)



Также можно создать проект в PyCharm (либо другой ide) и запускать из него.
